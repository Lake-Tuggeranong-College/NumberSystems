Decimal Conversion

Converting from Decimal to Binary (and other number systems) is needed for various reasons.

This page will convert the binary number you enter into both Decimal and Hexadecimal.

In the HTMl file, there will eventually be a number of functions (modules).
The button will call convertBinary() which prompt the user for the binary number through an prompt() command.

convertDecimal() will call convertToBinary(var decimalInput) and then call convertToHexadecimal(var decimalInput)
convertToBinary() - converts binaryInput into Decimal (base 10) and updates the relevant HTML tag.
convertToHexadecimal(var binaryInput) - converts binaryInput into Hexadecimal (base 16) and will return the value back to convertDecimal to update the relevant HTML.
